Proyecto: Animación frame por frame (Michael Jackson - Moonwalk)

Objetivo
Comprender y aplicar la técnica de animación cuadro por cuadro (frame by frame) utilizando una guía de poses (keyframes).

Para realizarlo, se necesitó de
Blender 5.0.1
Guía de poses (keyframes de referencia de moonwalk)

Desarrollo
Se utilizó una guía de poses (keyframes) de un personaje realizando el paso de baile "moonwalk". A partir de esta referencia, se dibujaron manualmente las distintas posiciones del personaje en cada cuadro.

Se empleó la técnica de "piel de cebolla" (onion skin) para visualizar cuadros anteriores y mantener coherencia en el movimiento.

El proceso consistió en:
1. Dibujar las poses clave principales
2. Completar los cuadros intermedios
3. Ajustar detalles para suavizar la animación

A diferencia de las prácticas anteriores, no se utilizó rigging ni scripting, sino que todo el movimiento fue construido manualmente cuadro por cuadro.

Resultados
Se obtuvo una animación continua del movimiento tipo moonwalk. El resultado mostró mayor fluidez y control artístico, aunque requirió más tiempo y precisión.

Conclusión
La animación frame por frame permite un control total sobre el movimiento, logrando resultados más expresivos y detallados. Sin embargo, es un proceso más laborioso en comparación con otras técnicas como rigging o scripting (aunque, preferible a gusto personal).
