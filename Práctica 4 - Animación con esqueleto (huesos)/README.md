Proyecto: Calco y animación con esqueleto (Steve de Minecraft)

Objetivo
Aprender a utilizar sistemas de rigging (esqueleto) en Blender para generar animaciones más realistas en personajes 2D.

Para esta práctica, se necesitó
Blender 5.0.1
Imagen de referencia (Steve de Minecraft)
Desarrollo

Se inició importando una imagen de Steve de Minecraft como referencia. Posteriormente, se calcó el personaje utilizando herramientas de dibujo, separando visualmente las partes importantes (cabeza, brazos, etc.) para facilitar la animación.

Después, se creó un sistema de huesos (armature) que correspondiera a las partes principales del cuerpo del personaje. Cada hueso fue vinculado a su respectiva parte del dibujo mediante asignación de pesos o parenting.

Una vez configurado el esqueleto, se realizaron pequeñas animaciones moviendo manualmente algunos huesos, específicamente:

Movimiento del brazo
Movimiento de la cabeza

Esto permitió observar cómo el dibujo respondía de manera más natural a los cambios de posición.

Resultados
Se obtuvo un personaje calcado con un sistema de huesos funcional, capaz de realizar movimientos parciales. A diferencia de la práctica anterior, el movimiento fue más flexible y realista.

Conclusión
La implementación de un esqueleto permitió mejorar significativamente la calidad de la animación. Se comprendió la importancia del rigging para animaciones más complejas y realistas, así como la relación entre huesos y mallas/dibujos.
