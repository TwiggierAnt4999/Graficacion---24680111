Proyecto: Calco y animación básica en Blender (Darth Vader)

Familiarizarse con el uso de herramientas de dibujo 2D en Blender, así como introducir conceptos básicos de animación mediante scripting.

Para realizarlo se necesitó de lo siguiente
- Blender 5.0
- Imagen de referencia (Darth Vader)
- Desarrollo

Primero, se importó una imagen de referencia de Darth Vader en Blender para utilizarla como guía. Utilizando herramientas de dibujo (Grease Pencil), se procedió a calcar la imagen, cuidando proporciones y detalles principales del personaje.

Una vez terminado el calco, se implementó una animación básica mediante un script en Python dentro de Blender. Este código permitió que el dibujo se desplazara en el plano al presionar las teclas de dirección del teclado (arriba, abajo, izquierda y derecha).

El movimiento generado fue completamente rígido, es decir, todo el dibujo se desplazaba como un solo objeto sin deformaciones ni articulaciones.

Resultados
Se obtuvo un dibujo calcado de Darth Vader correctamente posicionado y funcional dentro del entorno de Blender. Además, se logró implementar interacción básica mediante teclado, permitiendo mover el objeto en distintas direcciones.

Conclusión
Esta práctica permitió comprender el flujo básico de trabajo en Blender para dibujo 2D y una introducción al uso de scripts para generar animaciones simples. Se observó que este tipo de animación es limitada, ya que no permite movimientos realistas al no considerar articulaciones.
